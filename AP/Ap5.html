Projet École de musique (Client lourd)

Description du projet

Dans le cadre de cette activité professionnelle, j’ai été amené à poursuivre le développement d’un projet déjà existant : un site dynamique réalisé pour une école de musique. Ce site permet de présenter l’école, ses cours, ses activités, d’afficher un planning hebdomadaire dynamique des cours, et de gérer les inscriptions des adhérents en ligne.

Dans cette nouvelle phase, j’ai développé une application PC en client lourd destinée à l’administrateur du site, notamment la secrétaire de l’école. Cette application permet de gérer plus facilement le contenu du site, comme les informations sur les enseignants, les cours, les inscriptions et le planning.

L’application utilise la même base de données que celle du site, ce qui assure une synchronisation en temps réel entre les deux interfaces : les modifications effectuées depuis l’application sont immédiatement visibles sur le site web. Grâce à cette approche, l’administrateur peut piloter l’ensemble du fonctionnement du site directement depuis l’application, sans avoir besoin d’accéder à l’interface web. Cela permet une gestion centralisée, pratique et plus sécurisée des données de l’école.

Outils utilisés

Pour réaliser ce projet, j'ai utilisé plusieurs outils. Visual Studio a été utilisé pour le développement de l'application PC en client lourd. PhpMyAdmin a été utilisé pour la gestion de la base de données, permettant une administration facile et efficace des données du site et de l'application.
